float FaceRecognizer::matched_max_threshold = 0.4
static

The maximum similarity threshold that indicates a match.

This static member variable specifies the maximum similarity threshold that indicates a match. The default value is set to the threshold used by the dlib face recognition model, which achieves an accuracy of 99.38% on the LFW face recognition benchmark.
